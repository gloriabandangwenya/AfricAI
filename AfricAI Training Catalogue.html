Microsoft Excel
📊

Excel is the backbone of data-driven work across every industry in Africa. Our three-level Excel training takes participants from complete beginner through to confident data analyst — equipping them with practical, immediately applicable skills that transform how they work every day.

Level 1 — Foundations 6 hrs / Full Day
Designed For
New Excel users and professionals who rely on Excel daily but have never received formal training — admin staff, coordinators, entry-level analysts, and frontline teams.
What Participants Will Be Able To Do
  • Navigate Excel confidently — ribbons, views, workbooks, and worksheets
  • Enter, edit, and organise data accurately and efficiently
  • Use essential formulas: SUM, AVERAGE, COUNT, IF, and cell references
  • Format workbooks professionally for internal reports and client-facing documents
  • Sort, filter, and manage data tables for quick analysis
  • Create clear, well-labelled charts from data sets
  • Set up and print documents correctly for professional output
  • Understand how Copilot and AI tools connect to Excel basics

$75 per person
Level 2 — Intermediate 6 hrs / Full Day
Designed For
Users comfortable with Excel basics who want to work faster, analyse data more deeply, and reduce manual work — supervisors, analysts, finance staff, and team leads.
What Participants Will Be Able To Do
  • Use advanced lookup functions: VLOOKUP, HLOOKUP, XLOOKUP, and INDEX/MATCH
  • Apply logical functions: IF, AND, OR, nested IF statements
  • Work with named ranges for cleaner, more readable formulas
  • Build and customise PivotTables for instant data summarisation
  • Apply conditional formatting to highlight critical information visually
  • Use data validation to build controlled, error-free input forms
  • Create dynamic charts and sparklines for reporting
  • Leverage Copilot and AI tools within intermediate Excel workflows

Experienced Excel users who need to handle complex data, build automated systems, and create executive-level dashboards — senior analysts, finance managers, operations leads, and data teams.

  • Import, clean, and transform data using Power Query
  • Build complex nested and array formulas for advanced analysis
  • Design interactive executive dashboards with slicers and dynamic charts
  • Automate repetitive tasks using macros and basic VBA
  • Consolidate data across multiple sheets and workbooks
  • Apply advanced PivotTable techniques including calculated fields
  • Build data models and use Power Pivot for large datasets
  • Integrate Copilot for automated data analysis and insight generation
